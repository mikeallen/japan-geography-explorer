# Archived Data Files

This directory contains intermediate and outdated data files from the development process.

## File Descriptions

### japan_outline_geo.csv
Early version of Japan's outline coordinates. Superseded by prefecture boundary data.

### mountains_list.csv
Initial list of mountains before enhancement and consolidation.

### mountains_new.csv
Intermediate version of mountains data during the merging process.

### plains.csv
Plains data that was considered but not included in the final application.

### river_basins.csv
River basin data that was explored but not used in the final version.

### rivers_geo_cleaned.csv
Intermediate version of river geometry after initial cleaning.

### rivers_geo_new.csv
Another intermediate version during the river data processing pipeline.

### sake_rice_backup.csv
Backup of sake rice data before major enhancements (production data, importance ratings, etc.).

## Purpose

These files are retained for:
- Historical reference
- Data lineage documentation
- Potential future feature additions (e.g., plains, river basins)
- Ability to roll back to earlier data versions if needed

## Current Data Location

The active, production-ready data files are in the parent directory and include:
- prefectures.csv / prefectures_geo.csv
- old_provinces.csv / old_provinces_geo.csv
- mountains.csv / mountains_geo.csv
- rivers.csv / rivers_geo_final.csv
